Text redactor
Text redactor - это утилита, которая позволяет внутри себя выполнять : ввод текста, редактировать текст, делать выделенный текст обзацем, заголовком, добавлять изображения с помощью ввода URl адреса. В утилиту встроен кастомный локальный сторадж, который сохраняет все изменения и позволяет делать шаги назад и вперед. Также предусмотрена возможность копировать HTML разметку отредактрованного содержимого. Проект напасан с помощью :

Backend - нет, frontend lang - javasctipt, framework Vue 3, CSS3 Чем занимался на проекте :

Простой дизайн взятый из головы, вёрстка, минимальный UX привычный для большинства пользователей, использование Vue 3 как нового иснтрумента разработчки с его базовым изучением, написание логики на нативном JS с достаточно тесной работой с DOM-деревом напрямую. Проект можно потрогать в виде деплоя тут, а сам код в данном репозитории
